# 快捷运维：备案相关查询接入说明

## 背景

- **工信部公开备案检索**：以 [beian.miit.gov.cn](https://beian.miit.gov.cn/) 为准；阿里云**不提供**与其完全一致的「全文备案库」官方 OpenAPI 供第三方随意检索。
- **阿里云官方「备案」OpenAPI（Beian / 2016-08-10）**：SDK `beian20160810`，典型接口如 `QueryAccessorDomainStatus`，用于**接入/核验场景下的域名状态**（返回 `status`、`reason` 等），**不是**工信部备案证的完整展示。
- **阿里云云市场「ICP 备案查询」类 API**：多为第三方数据服务，通常使用 **AppCode** 调用，**按次或套餐计费**；需在云市场单独购买后配置 URL 与本平台环境变量。

本仓库在 Nexus「备案」页同时支持：

1. **官方查询**：使用当前快捷运维已保存的**云账号 AccessKey** 调用 `QueryAccessorDomainStatus`。
2. **云市场查询**（可选）：服务端配置 `nexus.icp.market.api-url` 与 `nexus.icp.market.app-code` 后，前端可触发服务端代发 HTTP 请求（避免将 AppCode 暴露给浏览器）。

## 配置

| 项 | 说明 |
|----|------|
| `NEXUS_ICP_MARKET_API_URL` | 云市场商品文档中的完整 HTTPS 地址。若文档要求路径含域名，可使用占位 `{domain}`，平台会替换为 URL 编码后的域名；否则在 URL 后自动追加 `?domain=`。 |
| `NEXUS_ICP_MARKET_APP_CODE` | 云市场分配的 AppCode，写入请求头 `Authorization: APPCODE <code>`。 |

对应 `application.properties`：

```properties
nexus.icp.market.api-url=${NEXUS_ICP_MARKET_API_URL:}
nexus.icp.market.app-code=${NEXUS_ICP_MARKET_APP_CODE:}
```

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/admin/nexus/accounts/{accountId}/icp/query-accessor-domain-status?domain=` | 官方 Beian：`QueryAccessorDomainStatus` |
| GET | `/api/admin/nexus/icp/market-query?domain=` | 云市场第三方（需上述配置） |

RAM：官方接口需为云账号 AK 授权 Beian 相关产品接口（以阿里云控制台策略为准）。

## 合规提示

云市场返回体可能包含主办单位等敏感信息，请遵守采购协议与《个人信息保护法》等，生产环境建议脱敏展示与访问审计（本平台已写 `nexus.icp.query.*` 审计日志）。
